Project Overview

Using html and css, I created a website similar to wikipedia, using multiple pages. It was developed to provide information for the comic book character Dennis the menace and his companion Gnasher. I included a home page, a about the character page, about the author page and the history of Dennis the Menace which was all linked through a style sheet with pictures to make all the webpages look simialr and professional.

My Role

My main role was to lead this project as I spent time organising what pages each person would create and then creating the stylesheet so all the pages would look the same and professional, I also created the home and about pages using HTML and CSS.
